# 使用介绍

## 商户端

### 概论

灵时云商户端操作手册是为灵活用工系统的使用者迅速了解系统，高效率高质量的使用该系统

### 功能

主要具有首页、任务管理、创客管理、结算管理、票税管理、财务管理、我的账户、API对接等功能，能够为商户提供更加便捷，全面的服务

### 商户端登录

打开官网[http://s.linsyun.com](http://s.linsyun.com)根据运营人员创建的商户账号进行登录，输入账号、密码，登录成功之后如下图：

![图片1.png](./img1.png)

商户端首页，查看该商户的账户余额、可用余额、冻结金额等信息。快捷导航迅速跳转到需要进入的功能页面。消息通知是围绕系统与用户进行沟通交流的重要方式，目的在于让用户了解新事件的信息动态，系统文档为让操作商户查看系统的操作文档，接口文档等。

![图片2.png](./img2.png)

### 任务管理

商户登陆后，点击侧边导航栏-任务管理，查看已发布的所有任务，可以输入任务名称、任务编号进行任务筛选如下图：

![图片3.png](./img3.png)

#### 发布任务

点击【发布任务】按钮，填写任务名称、任务内容，点击【确定】按钮后，任务自动生成任务编号，如下图：

![图片4.png](./img4.png)

#### 查看

点击查看按钮，可查看任务的基本信息和任务下的用工计划，如下图所示：

![图片5.png](./img5.png)

点击查看页面中的【参与创客】按钮，可查看该任务下的创客人员，如下图所示：

![图片6.png](./img6.png)

#### 终止

点击【终止】按钮，任务变为终止状态，同时可修改任务的基本信息，如下图：

![图片7.png](./img7.png)

### 创客管理

点击侧边导航栏-创客管理，查看所有已上传的创客。可以在筛选条件单独输入创客编号、姓名、手机号、证件号查找创客

也可以选择签约状态如未签约、已签约、签约失败进行创客的筛选，如下图：

![图片8.png](./img8.png)

### 结算管理

#### 结算单

点击侧边导航栏-结算管理-结算单，查看已上传的所有结算单，增加了对应的任务编号、任务名称，可以通过结算单编号、任务编号、任务名称、审核状态、发放状态、创建时间、付款渠道进行结算单查询，如下图：

![图片9.png](./img9.png)

#### 结算单导入

点击【结算单】按钮，进入结算单上传页面，需要先选择文件，然后才能导入结算单，如下图：（Web API用工上传结算单时，每个结算单系统会自动生成一条任务，任务名称：商户名称创建任务，任务描述：商户名称准备打款，任务金额：计划金额-计划金额*100）

![图片10.png](./img10.png)

#### 整单拒绝发放

进入结算单列表，针对审核通过的结算单，可以进行整单拒绝发放，如下图所示：

![图片11.png](./img11.png)

点击【拒绝发放】按钮，进行二次确认，点击确认后，整单拒绝发放，结算单状态变为拒绝发放，其下明细数据的发放状态变为拒绝发放，打款结果变为拒绝打款，如下图所示：

![图片12.png](./img12.png)

![图片13.png](./img13.png)

#### 部分拒绝发放 

进入结算单列表，审核通过的结算单，点击【查看】按钮，进入结算单详情界面，可对明细数据进行部分拒绝发放，如下图所示：

![图片14.png](./img14.png)

点击【拒绝发放】按钮，进行二次确认，点击确认后，可对当前的数据行进行拒绝发放，发放状态变为拒绝发放，打款结果变为拒绝打款，如下图所示：

![图片15.png](./img15.png)

![图片16.png](./img16.png)

### 结算明细

点击侧边导航栏-结算管理-结算明细，查看已上传的所有结算单的数据明细，汇总当前搜索状态下，打款结果各状态的汇总金额和汇总笔数，并可将明细数据导出，如下图：

![图片17.png](./img17.png)

### 我的账户

点击侧边导航栏-我的账户，查看该账户的总余额、可用余额、冻结金额。查看该账户的银行账户、支付宝账户信息：账户余额、冻结金额、账户名称（可复制）、账户号码（可复制）、开户银行（可复制）。如下图：

![图片31.png](./img31.png)

### 财务中心

#### 充值记录

点击侧边导航栏-财务管理-充值记录，查看该商户的充值记录,可以看到充值的打款时间、流水号、打款账号、打款户名、打款金额、充值状态、收款账号、收款户名、备注等详细信息，选择打款开始时间至打款结束时间、充值状态查询充值记录。如下图：

![图片29.png](./img29.png)

#### 支出流水

点击侧边导航栏-财务管理-支出流水，查看该商户的支出记录，选择打款开始时间至结束时间查询账户的支出记录。如下图：

![图片30.png](./img30.png)

### 票税管理

#### 发票申请

点击侧边导航栏-票税管理-发票申请，当前页面显示已发放完成的所有结算单，可以输入结算单号，选择发放日期进行发票的筛选如下图：

![图片18.png](./img18.png)

勾选可开票结算单后，点击【申请开票】按钮，进入申请开票流程，如下图所示：

![图片19.png](./img19.png)

点击下一步，如下图：

![图片20.png](./img20.png)

发票抬头、收件人信息模块，点击【更换】按钮，可更换对应的信息，如下图所示：

![图片21.png](./img21.png)

![图片22.png](./img22.png)

点击【提交】按钮，开票申请提交成功，等待审核，可在开票记录中查看到对应的开票申请详细信息。

#### 开票记录

点击侧边导航栏-票税管理-开票记录，当前页面显示已提交的所有开票申请，输入申请编号或者选择审核状态、开票状态进行开票记录的筛选如下图：

![图片23.png](./img23.png)

点击【查看】按钮，可查看详细信息，如下图所示：

![图片24.png](./img24.png)

#### 发票抬头

点击侧边导航栏-票税管理-发票抬头，当前页面显示用户添加的所有抬头信息，输入抬头名称、电话号码进行查询。如下图：（用户申请开票的时候，系统将自动填充默认的抬头信息）

![图片25.png](./img25.png)

点击【新增】按钮可以新增抬头信息，如下图：

![图片26.png](./img26.png)

点击【编辑】按钮可以修改抬头信息；

点击【默认】按钮可将当前抬头信息设置为默认；

点击【删除】按钮可以删除当前抬头信息。

#### 邮寄地址

点击侧边导航栏-票税管理-邮寄地址，当前页面显示用户添加的所有地址信息，输入收件人姓名、收件人电话来查询收件人。如下图：（用户申请开票的时候，系统将自动填充默认的地址信息）

![图片27.png](./img27.png)

点击【新增】按钮可以新增地址信息，如下图：

![图片28.png](./img28.png)

点击【编辑】按钮可以修改地址信息；

点击【默认】按钮可将当前地址信息设置为默认；

点击【删除】按钮可以删除当前地址信息。

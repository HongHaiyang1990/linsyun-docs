    # 创客接口

    # 1. 创客签约

    为实现结算的顺利开展，需在结算前提供以必要的信息和完成必要流程：

    • 商户收集创客资料（姓名、身份证、手机号），通过创客上传接口上传至灵时云
    • 调用创客查询接口，获取创客签约状态&签约链接；未签约状态下，可使用签约链接进行签约，或者登录灵时云系统-创客列表，点击签约按钮，手动给创客发送签约短信；签约完成后，创客签约状态更新，可通过创客查询接口查询

    # 1.1 创客上传接口

    请求地址:

    https://{BASE_URL}/api/WorkPeople/uploadWorkPeoples

    请求方式: POST

    BASE_URL:

    环境 地址
    测试环境 pre-lsy-outsource-openapi-interface-api.suuyuu.cn
    正式环境 o.linsyun.com

    请求参数 请求参数以JSON方式提交

    参数名 参数说明 长度和属性 必填 备注
    memberCode 商户号 String(50) √ 商户号，参与签名；例如：SH123456
    updateMarkNo 上传标识编号 String(50) √ 上传表示编号，需要全局唯一，参与签名；例如：A001
    timestamp 时间戳 String(20) √ 时间戳，格式为：yyyy-MM-dd HH:mm:ss，与当前时间误差不能超过10分钟，参与签名；例如：2000-01-01 10:10:10
    sign 签名 String(50) √ 签名规则，见文档中说明；例如：D1BD444BFC916C414248CB9BDC5300561208AD03
    workPeoples 创客信息 List<RequestUploadWorkPeopleModel> √ 创客信息集合

    RequestUploadWorkPeopleModel

    参数名 参数说明 长度和属性 必填 备注
    userName 姓名 String(50) √
    idCard 身份证号 String(50) √
    mobile 手机号 String(50) √
    email 电子邮箱 String(200)
    sex 性别 Integer 1：男，2：女
    address 地址 String(200)
    idFrontPhoto 身份证正面图片 String(200) 图片网络地址
    idBackPhoto 身份证反面图片 String(200) 图片网络地址
    bankAccount 银行卡号 String(50)
    alipayAccount 支付宝账号 String(50)

    响应参数

    参数名 参数说明 长度和属性 必填 备注
    code 状态码 String √ 正常：0，详情请查看【状态码说明】
    message 返回信息描述 String √
    messageType 返回信息类型 Integer √
    data 返回数据 List<ResponseUploadWorkPeopleModel> 校验不通过信息

    ResponseUploadWorkPeopleModel

    参数名 参数说明 长度和属性 必填 备注
    idCard 身份证号 String √ 校验不通过的身份证号
    remark 备注 String √ 校验不通过原因

    # 1.2 创客查询接口

    请求地址:

    https://{BASE_URL}/api/WorkPeople/QueryWorkPeople

    请求方式: POST

    BASE_URL:

    环境 地址
    测试环境 pre-lsy-outsource-openapi-interface-api.suuyuu.cn
    正式环境 o.linsyun.com

    请求参数 请求参数以JSON方式提交

    参数名 参数说明 长度和属性 必填 备注
    memberCode 商户号 String(50) √ 商户号，参与签名；例如：SH123456
    updateMarkNo 上传标识编号 String(50) √ 上传表示编号，需要全局唯一，参与签名；例如：A001
    timestamp 时间戳 String(20) √ 时间戳，格式为：yyyy-MM-dd HH:mm:ss，与当前时间误差不能超过10分钟，参与签名；例如：2000-01-01 10:10:10
    sign 签名 String(50) √ 签名规则，见文档中说明；例如：D1BD444BFC916C414248CB9BDC5300561208AD03
    seachData 查询信息 RequestQueryWorkPeopleModel √ 查询创客信息

    RequestQueryWorkPeopleModel

    参数名 参数说明 长度和属性 必填 备注
    userName 姓名 String(50)
    idCard 身份证号 String(50)

    响应参数

    参数名 参数说明 长度和属性 必填 备注
    code 状态码 String √ 正常：0，详情请查看【状态码说明】
    message 返回信息描述 String √
    messageType 返回信息类型 Integer √
    data 返回数据 ResponseQueryWorkPeopleModel 数据信息

    ResponseQueryWorkPeopleModel

    参数名 参数说明 长度和属性 必填 备注
    list 创客数据集合 List<ResponseWorkPeopleModel> 创客信息集合
    statistics 集合信息 ResponseWorkPeopleModel 创客信息集合

    ResponseWorkPeopleModel

    参数名 参数说明 长度和属性 必填 备注
    userName 姓名 String √
    idCard 身份证号 String √
    mobile 手机号 String √
    email 电子邮箱 String
    sex 性别 Integer 1：男，2：女
    address 地址 String
    idFrontphoto 身份证正面 String 正面图片网络地址
    idBackphoto 身份证反面 String 反面图片网络地址
    signState 签约状态 Integer √ 0：未签约，1：已签约，2：签约失败
    signFailCause 签约失败原因 String
    signUrl 签约链接 String √ 签约网络链接

    参数示例

    # 请求参数示例
{
    "memberCode": "SH123456",
    "updateMarkNo": "A001",
    "timestamp": "2000-01-01 10:10:10",
    "sign": "D1BD444BFC916C414248CB9BDC5300561208AD03",
    "searchData": {
        "userName": "张三",
        "idCard": "11010120000101001X"
    }
}

    # 2. 状态码说明

    CODE值 说明
    0 成功
    999 失败
    1000 缺少必要参数
    1001 时间戳格式错误
    1002 时间戳已超过有效期
    1003 上传中请勿重复提交
    1004 商户不存在
    1005 商户配置异常
    1006 签名错误
    1007 上传明细数据超过2000条
    1008 上传标识编号已存在
    1009 未上传数据
    1017 存在未校验通过的数据
    1018 明细数据存在空商户流水号的数据
    1019 全订单明细未上传成功
    1020 未开通支付宝通道，下发类型错误
    1021 未开通银行通道，下发类型错误
    1022 任务不存在
    1023 任务未审核通过
    5000 系统异常

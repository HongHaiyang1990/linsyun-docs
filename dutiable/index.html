        # 完税接口

        # 1.完税查询接口

        # 1.1 完税凭证查询接口

        请求地址:

        http://{BASE_URL}/api/Taxation/QueryTaxPaymentReceipt

        请求方式: POST

        BASE_URL:

        环境 地址
        测试环境 pre-lsy-outsource-openapi-interface-api.suuyuu.cn
        正式环境 o.linsyun.com

        请求参数：

        参数名 参数说明 长度和属性 必填 备注
        memberCode 商户号 String √ 商户号，参与签名；例如：9244924
        updateMarkNo 上传标识编号 String(50) √ 上传标识编号，需要全局唯一，参与签名；例如：A11000001
        timestamp 时间戳 String(50) √ 格式为：yyyy-MM-dd HH:mm:ss，与服务器的当前时间误差不能超过10分钟，参与签名；例如：2020-11-19 16:51:20
        sign 签名 Varchar(50) √ 签名规则，见文档中说明；例如：D1BD444BFC916C414248CB9BDC5300561208AD03
        DateMonth 月份 Varchar(50) √ 查询月份，例如：2023-01

        响应返回结构

        参数名 参数说明 长度和属性 必填 备注
        code 状态码 Varchar(10) √ 0:正常，10xx:客户端异常同上，5000:服务器异常
        message 返回信息描述 Varchar(50) √
        messageType 返回信息类型 Int √

        none = 0,info = 1,warn = 2,

        error = 3,success = 4

        + data 返回数据 List<TaxPaymentReceipt> ☆

        -taxVoucherNo

        		 完税凭证编号 Varchar(255) √
        -taxVoucherName 完税凭证名称 Varchar(255) √
        -supplierName 完税税源地名称 Varchar(255) √
        -taxVoucherDateMonth 完税月份 Varchar(255) √
        -receipts 凭证附件列表 List<ReceiptInfo> √
        --fileName 凭证附件名称 Varchar(255) √
        -fileUrl 凭证附件地址 Varchar(255) √ 网络地址

        参数示例

        # 2. 状态码说明

        CODE值 说明
        0 成功
        999 失败
        1000 缺少必要参数
        1001 时间戳格式错误
        1002 时间戳已超过有效期
        1003 上传中请勿重复提交
        1004 商户不存在
        1005 商户配置异常
        1006 签名错误
        1007 上传明细数据超过2000条
        1008 上传标识编号已存在
        1009 未上传数据
        1017 存在未校验通过的数据
        1018 明细数据存在空商户流水号的数据
        1019 全订单明细未上传成功
        1020 未开通支付宝通道，下发类型错误
        1021 未开通银行通道，下发类型错误
        1022 任务不存在
        1023 任务未审核通过
        5000 系统异常
        5000 系统异常

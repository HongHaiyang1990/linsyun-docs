首页，如下图：

# 任务管理

商户登陆后，点击侧边导航栏-任务管理，查看已发布的所有任务，如下图：

# 发布任务

点击【发布任务】按钮，填写任务名称、任务内容，点击【确定】按钮后，任务生成，如下图：

# 查看

点击查看按钮，可查看任务的基本信息和任务下的用工计划，如下图所示：

点击查看页面中的【参与创客】按钮，可查看该任务下的创客人员，如下图所示：

# 终止

点击【终止】按钮，任务变为终止状态，同时可修改任务的基本信息，如下图：

# 创客管理

点击侧边导航栏-创客管理，查看所有已上传的创客，如下图：

# 结算管理

# 结算单

点击侧边导航栏-结算管理-结算单，查看已上传的所有结算单，增加了对应的任务编号、任务名称，如下图：

# 结算单导入

点击【结算单】按钮，进入结算单上传页面，需要先选择文件，然后才能导入结算单，如下图：（Web API用工上传结算单时，每个结算单系统会自动生成一条任务，任务名称：商户名称创建任务，任务描述：商户名称准备打款，任务金额：计划金额-计划金额*100）

# 整单拒绝发放

进入结算单列表，针对审核通过的结算单，可以进行整单拒绝发放，如下图所示：

点击【拒绝发放】按钮，进行二次确认，点击确认后，整单拒绝发放，结算单状态变为拒绝发放，其下明细数据的发放状态变为拒绝发放，打款结果变为拒绝打款，如下图所示：

# 部分拒绝发放

进入结算单列表，审核通过的结算单，点击【查看】按钮，进入结算单详情界面，可对明细数据进行部分拒绝发放，如下图所示：

点击【拒绝发放】按钮，进行二次确认，点击确认后，可对当前的数据行进行拒绝发放，发放状态变为拒绝发放，打款结果变为拒绝打款，如下图所示：

# 结算明细

点击侧边导航栏-结算管理-结算明细，查看已上传的所有结算单的数据明细，汇总当前搜索状态下，打款结果各状态的汇总金额和汇总笔数，并可将明细数据导出，如下图：

# 票税管理

# 发票申请

点击侧边导航栏-票税管理-发票申请，当前页面显示已发放完成的所有结算单，如下图：

勾选可开票结算单后，点击【申请开票】按钮，进入申请开票流程，如下图所示：

点击下一步，如下图：

发票抬头、收件人信息模块，点击【更换】按钮，可更换对应的信息，如下图所示：

点击【提交】按钮，开票申请提交成功，等待审核，可在开票记录中查看到对应的开票申请详细信息。

# 开票记录

点击侧边导航栏-票税管理-开票记录，当前页面显示已提交的所有开票申请，如下图：

点击【查看】按钮，可查看详细信息，如下图所示：

# 发票抬头

点击侧边导航栏-票税管理-发票抬头，当前页面显示用户添加的所有抬头信息，如下图：（用户申请开票的时候，系统将自动填充默认的抬头信息）

点击【新增】按钮可以新增抬头信息，如下图：

点击【编辑】按钮可以修改抬头信息；

点击【默认】按钮可将当前抬头信息设置为默认；

点击【删除】按钮可以删除当前抬头信息。

# 邮寄地址

点击侧边导航栏-票税管理-邮寄地址，当前页面显示用户添加的所有地址信息，如下图：（用户申请开票的时候，系统将自动填充默认的地址信息）

点击【新增】按钮可以新增地址信息，如下图：

点击【编辑】按钮可以修改地址信息；

点击【默认】按钮可将当前地址信息设置为默认；

点击【删除】按钮可以删除当前地址信息。

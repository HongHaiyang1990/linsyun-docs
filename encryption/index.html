      # 签名说明

      # 签名步骤

      1. 接口请求参数中 memberCode、updateMarkNo、timestamp参与签名
      2. 将所有参与签名字段按照参数名的字典顺序排序
      3. 将参数与值按照顺序拼接，列如：a=1、c=3、b=2 变为 a1b2c3
      4. 将密钥作为参数appSecret={key}，按第3步规则拼接到排序字符串中，列如：参与签名字段是a1b2c3，密钥字段appSecret{key}， 则变为appSecret{key}a1b2c3，这时我们定义他为signStr。（signStr=appSecrt+{key}+a1b2c3）；
      5. 将key+排序后的参数+secret,得到加密前字符串，此时的字符串为：key+signStr+secret；
      6. 对加密前的字符串进行 SHA1 加密，然后转为大写字符串，得到签名；
      7. 将得到的签名值请求参数sign；

      # 签名示例

      以请求入参示例中的数据做出签名
